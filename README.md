# dkz-scolia-overlay
Scoliadarts.com Overlay für z.B. OBS in Stil der offiziellen PDC Übertragungen

## Nutzung

Die bereitgestellten Dateien (Overlay-Pages und zugehörige Dateien) dürfen frei verwendet werden, 
um das Overlay zu starten, konfigurieren und zu nutzen.

Eine Veränderung, Weiterverbreitung oder eigene Veröffentlichung der Dateien 
ist ohne meine ausdrückliche Zustimmung nicht gestattet.

Den aktuellen Download findest du unter [Releases](https://github.com/dekayzzz/dkz-scolia-overlay/releases/).
