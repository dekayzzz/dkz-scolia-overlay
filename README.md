## Features

- **TV-Style Scoreboard** - Professionelle Darts Overlays im PDC-Stil für deine Streams
- **Auto-Zoom** - Automatisches Zoomen auf 180 und Doppel während des Werfens
- **OBS Browser Source** - Einfache Integration in OBS Studio
- **6 Build-In Presets** - Sofort einsatzbereite Designs
- **Komplett anpassbar** - Farben, Fonts, Layouts, Statistiken individuell konfigurierbar
- **Mini-Node-Webserver** - Alles Lokal auf deinem PC
- **OBS Automation** - Automatischer Scene Switch und Quellen-Steuerung

## Download & Installation

### 1. Download

Den aktuellen Download findest du unter **[Releases](https://github.com/dekayzzz/dkz-scolia-overlay/releases/latest)**.

### 2. Starten

1. Die `.exe` Datei herunterladen
2. DKZ Scolia Tool installieren
3. Starten - fertig und schon kannst du dir dein Overlay anpassen.

### 3. In OBS einbinden

1. In OBS eine **Browser Source** hinzufügen
2. URL eingeben: `http://localhost:1337/overlay/compact`
3. Breite: `1920`, Höhe: `1080`
